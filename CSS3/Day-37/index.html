Math Functions

Math.abs(x) returns the absolute (positive) value of x

Math.ceil(x) rounds x up to its nearest integer

Math.floor(x) rounds x down to its nearest integer

Math.max(x, y, z, ..., n) returns the number with the highest value

Math.min(x, y, z, ..., n) returns the number with the lowest value

Math.pow(x, y) returns the value of x to the power of y

Math.random() returns a random number between 0 and 1

Math.round(x) rounds x to its nearest integer

Math.sqrt(x) returns the square root of x

Date Functions

Date.now() returns the number of milliseconds since January 1, 1970

Date.parse(date) returns the number of milliseconds since January 1, 1970, 00:00:00 UTC

Date.UTC(year, month, day, hours, minutes, seconds, milliseconds) returns the number of milliseconds since January 1, 1970, 00:00:00 UTC

Date() creates a new date object with the current date and time

Date(year, month, day, hours, minutes, seconds, milliseconds) creates a new date object with the specified date and time

Date(milliseconds) creates a new date object as zero time plus milliseconds

Date(date string) creates a new date object as zero time plus time value string
